이름 : cooking1.jpg
제목 : 뚝딱 완성 참치전
소제목 : 뚝딱 완성 참치전
참치캔만 있으면 반은 완성되는 요리.
참치캔과 부침가루 섞고,
취향에 따라 원하는 채소를 함께 넣고 부치면 완성!
인분 수 : 2인분
준비시간 : 10분
조리시간 : 5분
주재료 : 참치캔 1개(150g), 부침가루 1컵(150g)
부재료 : 청양고추 1개(10g), 홍고추 1/2개(5g), 쪽파 2줄기(20g)
양념 : 포도씨유 3스푼(30g), 물 1컵(180mL)
Step1. 재료 준비하기 : 쪽파와 청양고추, 홍고추는 다져서 준비해주세요.
(참치 통조림 속 기름은 요리에 사용가능해요!)
Step2. 반죽만들기 : 볼에 모든 재료를 넣고 골고루 섞어 주세요.
(부침가루 대신 밀가루로 대체 가능해요)
Step3. 전 부치기 : 예열된 팬에 포도씨유를 두른 후, 적당한 크기고 올려 중불로 앞뒷면
노릇하게 부쳐주면 완성!


이름 : cooking2.jpg
제목 : 별미간식 옥수수전
소제목 : 옥수수캔으로 고소하고 달콤한 이색 전
라이스페이퍼 한 장 올려주면 흩어질 걱정 없이
바삭하게 옥수수전 완성!
인분 수 : 2인분
준비시간 : 10분
조리시간 : 20분
주재료 : 옥수수캔 1개(340g), 튀김가루 1컵(120g)
부재료 : 티아시아 베트남 월남쌈 라이스페이퍼 3장(3g)
양념 : 포도씨유 4스푼(40g), 물 약 1컵(150mL), 소금 약간 
Step1. 옥수수캔 물기 제거하기 : 옥수수 캔은 채에 밭쳐 물기를 제거해 주세요.
Step2. 반죽 만들기 : 튀김가루와 물, 소금 섞어 반죽한 후, 준비한 옥수수를 넣고 섞어 주세요.
(튀김가루 대신 밀가루나 부침가루로 대체 가능해요)
Step3. 전 부치기 : 중불로 예열한 팬에 포도씨유를 넣고 라이스페이퍼를 깐 후, 옥수수전 반죽 한 국자를
떠서 올려주세요. 라이스페이퍼가 노릇해지면 뒤집어 주고, 앞 뒤가 노릇해질 때까지 뒤집개를
활용해 뒤집어 가며 구워주면 완성!(연유나 설탕, 파마산 치즈가루를 곁들어 먹을 수 있어요) 



이름 : cooking3.jpg
제목 : 불맛 가득 계란볶음밥
소제목 : 스크램블한 계란에 고슬고슬 밥 볶기
계란볶음밥이 항상 축축했던 사람은 주목!
불맛이 가미된 계란볶음밥 도전!
인분 수 : 1인분
준비시간 : 10분
조리시간 : 10분
주재료 : 밥 1공기(200g), 계란 1개(50g)
부재료 : 양파 1/6개(30g), 당근 1/6개(20g), 쪽파 2줄기(20g)
양념 : 연두진 1스푼(10g), 포도씨유 2스푼(20g)
Step1. 재료 준비하기 : 쪽파는 0.1cm 송송 썰어주고, 당근과 양파는 0.5cm 두께로 다져 주세요.
계란은 골고루 풀어 준비해 주세요.
Step2. 계란 스크램블하기 : 예열된 팬에 포도씨유를 두르고, 풀어둔 계란을 스크램블로 조리해
주세요. 계란이 절반정도 익었을 때, 연두진을 팬 가장자리에 넣어 그을려 향을 입혀 주세요.
Step3. 밥과 채소 넣어 볶기: 조리중인 팬에 밥을 넣고 부셔가며 볶아주고, 썰어둔 채소
(당근, 양파, 쪽파)를 넣고 볶아주면 완성!





이름 : cooking4.jpg
제목 : 밥도둑 새우장
소제목 : 짜지 않고 먹기 편한 깐새우장 꿀팁
간장과 물 1:2로 섞고
취향껏 매콤한 채소와 함께 생새우 넣어주면 완성!
인분 수 : 2인분
준비시간 : 10분
조리시간 : 20분
주재료 : 새우 6-8마리(180g)
부재료 : 꽈리고추 2개(10g), 홍고추 1/4개(5g), 편생강 1개(3g), 통마늘 2개(10g)
양념 : 물 1컵(200mL), 진간장 1/2컵(100g), 설탕 3.5스푼(35g)
Step1. 새우 준비하기 : 머리와 껍질을 제거한 새우는 이쑤시개를 이용하여 등쪽의 내장을 제거해 주세요.
Step2. 양념 간장 만들기 : 냄비에 물 1컵(200ml), 간장 1/2컵(100ml), 설탕(3.5스푼), 고추 ,생강, 마늘을
널고 중약불에서 끓여주세요. 간장물이 한 번 끓어 오르면 불을 꺼주고 용기에 담아 얼음물에 담긴 볼에
올려 식혀주세요.
Step3. 양념 간장 붓기 : 양념 간장의 열기가 빠지면 새우가 충분히 잠기도록 부어주세요. 밀폐용기에 담아
냉장고에서 숙성시키면 완성!(약 3시간 가량 뒤에 섭취 가능해요!)



이름 : cooking5.jpg
제목 : 별미간식 인절미
소제목 : 전자레인지로 조리해 손쉽게 만드는 법
집에서 만들어 따끈할 때 바로 먹어보아요.
인절미로 먹어도 좋고, 와플팬에 눌러 이색 디저트로 즐겨도 좋아요!
인분 수 : 2인분
준비시간 : 0분
조리시간 : 20분
주재료 : 찹쌀가루(습식) 1컵(200g)
부재료 : 따뜻한 우유 1/2컵(100g), 설탕 1.5스푼(15g), 소금 2꼬집(2g)
양념 : 볶은 콩가루 1/4컵(50g), 설탕 2스푼(20g), 소금 1꼬집(1g)
Step1. 우유에 밑간 하기 : 따뜻한 우유에 설탕 1.5스푼과 소금 2꼬집을 넣어 녹여 주세요.
Step 2. 반죽 만들고 익히기 : 체에 곱게 내린 찹쌀가루를 볼에 넣고 준비한 우유를 3~4번에 
나누어 부은 후, 덩어리지지 않게 섞어 주세요. 반죽에 랩을 씌우고 구멍을 뚫어 전자레인지에
1분 30초간 돌려 주세요. 다시 한 번 섞어준 뒤, (동일한) 랩을 다시 씌워 1분간 돌려주세요.
Step 3. 콩가루에 반죽 묻히기 : 볶은콩가루에 설탕과 소금을 넣어 섞은 뒤 넓은 바닥에 
넉넉하게 깔아 주세요. 준비한 반죽이 식기 전에 콩가루에 묻혀주면 인절미 완성!



이름 : cooking6.jpg
제목 : 길쭉길쭉 팽이버섯전
소제목 : 잘게 찢은 팽이버섯 위에 계란물을 부어 초간단 완성!
바삭 고소하게 부쳐주면
맛에 한 번, 식감에 한 번 놀랄거에요.
게살처럼 결이 살아있는 팽이버섯을 맛있게 먹어보아요!
인분 수 : 2인분
준비시간 : 5분
조리시간 : 5분
주재료 : 팽이버섯 1봉(120g), 달걀 2개(100g)
부재료 : 실파 2줄기(8g)
양념 : 연두순 1스푼(10g), 포도씨유 1/2스푼(6g)
Step 1. 재료 준비하기 :
팽이버섯은 아래 균사체 부분을 잘라낸 후, 뭉친 부분은 뜯어주세요.
실파는 1cm길이로 송송 썰어 주세요.
Step 2. 달걀 풀어 간하기 : 볼에 달걀과 연두순을 섞어주세요.
Step 3. 팽이버섯에 달걀 둘러 부치기: 달궈진 팬에 포도씨유를 두르고, 뜯어낸 팽이버섯을 
동그랗게 깔아주세요. 팽이버섯 위에 달걀물을 골고루 잘 붓고 실파를 올려 앞뒤 노릇하게 잘 부쳐주면 완성!
(팽이버섯 본연의 맛이 느끼하게 느껴진다면 후추를 조금 첨가하면 좋아요.)
